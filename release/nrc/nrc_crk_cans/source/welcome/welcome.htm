Start Using ᓀᐦᐃᔭᐍᐏᐣ (Cree Syllabics)

This keyboard is for typing nêhiyawêwin/Plains Cree/Cree Y-Dialect in syllabics (ᒐᐦᑭᐯᐦᐃᑲᓇ) on smartphones (e.g., Android/iPhone).

Tutorial

This ᓀᐦᐃᔭᐍᐏᐣ (Cree Syllabics) keyboard is a phonetic keyboard. That is, you type the way you say it in Cree.

For example, the word ᐁᑿ: it's spoken as ᐁ + ᐠ + ᐤ + ᐊ

To type ᐁᑿ on this keyboard, first press ᐁ:

The first sound in “ᑿ” is ᐠ, so next press ᐠ:

The keyboard has changed, and now the vowels (ᐁ ᐃ ᐅ ᐊ ᐄ ᐆ ᐋ) have been replaced by all syllables starting with the ᐠ sound (ᑫ ᑭ ᑯ ᑲ ᑮ ᑰ ᑳ, respectively).

But we don't want to type ᐁᑲ; we want to type ᐁᑿ! We need to add the dot that makes the ᐤ sound. To do that, press ᐤ. After pressing this, the keys will have changed yet again, with all of the syllables being replaced with syllables that start with the sounds ᐠᐤ, namely, ᑵ ᑷ ᑻ ᑿ ᑹ ᑽ ᒁ:

Finally, we press the syllable we want: ᑿ
